
# Anatomy & Physiology 120Q Online Quiz (PWA)

This folder contains a mobile-friendly, offline-capable web app (PWA).
Deploy it anywhere that serves static files (GitHub Pages, Netlify, Vercel).

## Quick Deploy: GitHub Pages
1) Create a new repo (e.g., `ap-quiz`).
2) Upload the *contents* of this folder (index.html, manifest.webmanifest, sw.js, icons/).
3) In **Settings → Pages**, choose "Deploy from a branch", branch `main`, folder `/ (root)`.
4) The site will be live at `https://<your-username>.github.io/<repo>/`.

## Quick Deploy: Netlify (drag & drop)
1) Go to https://app.netlify.com/drop
2) Drag this folder (or the ZIP) to deploy. Netlify will give you a URL.

## Add to Home Screen
- Android (Chrome): Open the URL → ⋮ → Add to Home screen.
- iOS (Safari): Open the URL → Share → Add to Home Screen.
